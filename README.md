# crypt-bde

Managing bitlocker on windows using golang, with escrow to [crypt-server](https://github.com/grahamgilbert/crypt-server)

## Building cryptbde

```golang
go get github.com/bdemetris/crypt-bde
```

```shell
make deps
```

```shell
make build
```
